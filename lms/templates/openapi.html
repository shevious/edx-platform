오픈 API

K-MOOC를 위한 안드로이드, 아이폰 등의 스마트 폰 기반의 앱개발을 희망하시는 개인, 기관, 기업은 정해진 절차에 따라 앱을 개발하실 수 있도록 기술문서를 공개합니다.

절차
##
제4조 (이용계약)
##
'서비스' 이용은 '당 홈페이지'가 허락하고 '회원'이 약관 내용에 대해 동의하면 됩니다.
##
제5조 (이용신청)
이용약관 확인 및 신청
- 신청서를 다운로드 해서 이용약관 동의 및 신청서를 작성하신 후 help_kmooc@nile.or.kr로 송부합니다
- 제목에 [오픈 API]를 기재하여 보내주세요.
  ○ 신청서 다운로드
오픈 API 문서
- 오픈 API 문서는 다음의 링크에서 다운로드 받으실 수 있습니다.
  ○ 오픈 API 다운로드
키발급
- 키발급은 승인 후 신청서에 기재된 메일 주소로 발급해드립니다.
문의사항
- 기술지원 이나 기타 문의 사항은 help_kmooc@nile.or.kr로 문의주세요.
- 제목에 [오픈 API]를 기재하여 보내주세요.
